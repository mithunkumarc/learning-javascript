#### Arrow functions do NOT have their own this.

Regular function:

        const obj = {
          name: "Cat",
          speak() {
            console.log(this.name); // "Cat"
          }
        };

        obj.speak(); 
Works because this is bound to obj.  

An arrow function inherits this from the surrounding (lexical) scope.
A regular function gets its this from how it is called.
        
        const obj = {
          name: "Cat",
          speak: () => {
            console.log(this.name); // NOT "Cat"
          }
        };
        obj.speak();

Instead, it refers to whatever this was outside the object when it was created (often the global object in browsers).
