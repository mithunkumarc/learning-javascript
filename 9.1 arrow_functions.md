function expression can be turned to arrow function/lambda funtion in python

        let func = function(arg1, arg2, ...argN) {
          return expression;
        };

        arrow function : 
        
        let func = (arg1, arg2, ...argN) => expression


Example : 

        let sum = (a, b) => a + b;
        alert( sum(1, 2) ); // 3
